Trading Application

A modern, extensible desktop trading platform built with Java 21 and Spring Boot, featuring real-time candlestick charting and WebSocket streaming from Alpaca Markets.

Java 21 Spring Boot WebSocket JFreeChart Reactive H2 Database

Key Features:

  • Real-time market data streaming via WebSocket
  • Interactive candlestick charts with multiple timeframes
  • Provider abstraction pattern for multi-broker support
  • Reactive non-blocking I/O with Spring WebFlux
  • H2 database persistence with Spring Data JPA
  • Componentized Swing UI with thread-safe data binding
